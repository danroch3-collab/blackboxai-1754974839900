Sobre Mí

Conoce mi trayectoria y pasión por el desarrollo Python

Soy un desarrollador Python apasionado por crear soluciones tecnológicas innovadoras. Mi experiencia abarca desde el desarrollo web con frameworks modernos hasta el análisis de datos y la automatización de procesos.

Me especializo en el desarrollo backend con Flask, Django y FastAPI, creando APIs robustas y aplicaciones web escalables. También tengo experiencia en análisis de datos con Pandas, NumPy y visualización con Matplotlib y Plotly.
